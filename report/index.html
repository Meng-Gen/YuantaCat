經營能力

杜邦分析

  • 股東權益報酬率 = 稅後純益 / 股東權益 = 純益率 x 總資產週轉率 x 權益乘數 = 資產報酬率 x 權益乘數
    • 企業為股東賺錢的投資報酬率
  • 資產報酬率 = 稅後純益 / 資產 = 純益率 x 總資產週轉率
    • 企業利用資產創造的收益
  • 純益率 = 稅後純益 / 營業收入
    • 代表每一元營收可以賺到的淨利
  • 資產週轉率 = 營業收入 / 資產
    • 代表資產利用率，力求產能滿載
  • 權益乘數 = 資產 / 股東權益
    • 代表槓桿，數值越大風險可能越高

財務結構

資產負債表分析重點

  • 自有資本比率 = 股東權益 / 資產
  • 負債比率 = 負債 / 資產
    • 觀察企業借貸狀況
  • 權益乘數 = 資產 / 股東權益

長期資金占固定資產比率

  • 長期資金占固定資產比率 = (長期負債 + 股東權益) / 固定資產
  • 看長期資金是否充足
