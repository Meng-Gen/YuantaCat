經營能力

杜邦分析

  • 股東權益報酬率 = 稅後純益 / 股東權益 = 純益率 x 總資產週轉率 x 權益乘數 = 資產報酬率 x 權益乘數
    • 企業為股東賺錢的投資報酬率
  • 資產報酬率 = 稅後純益 / 資產 = 純益率 x 總資產週轉率
    • 企業利用資產創造的收益
  • 純益率 = 稅後純益 / 營業收入
    • 代表每一元營收可以賺到的淨利
  • 資產週轉率 = 營業收入 / 資產
    • 代表資產利用率，力求產能滿載
  • 權益乘數 = 資產 / 股東權益
    • 代表槓桿，數值越大風險可能越高

財務結構

資產負債表分析重點

  • 自有資本比率 = 股東權益 / 資產
  • 負債比率 = 負債 / 資產
    • 觀察企業借貸狀況
  • 權益乘數 = 資產 / 股東權益
  • 長期資金占固定資產比率
    • 長期資金占固定資產比率 = (長期負債 + 股東權益) / 固定資產
    • 看長期資金是否充足

償債能力

流動比率 = 流動資產 / 流動負債

  • 數值越高代表流動資金越充足

速動比率 = 速動資產 / 流動負債

  • 進一步觀察流動性較高的資產

利息保障倍數 = 稅前息前淨利 / 利息費用

  • 越高代表長期貸款償債能力佳

獲利能力

損益表分析重點

  • 毛利率 = 營業毛利 / 營業收入
    • 毛利率越高產品競爭力越強
  • 營業利益率 = 營業利益 / 營業收入
    • 代表企業持續賺錢的能力
  • 稅前純益率 = 稅前純益 / 營業收入
    • 排除所得稅干擾，更貼近公司經營績效
  • 稅後純益率 = 稅後純益 / 營業收入
    • 影響股東權益

每月營業收入分析重點

  • 累積營收年增率
  • 長期平均線 (12ma)
  • 短期平均線 (3ma)

五大盈餘指標

  • 存貨指標 = 存貨增加率 - 營業收入增加率
  • 應收帳款指標 = 應收帳款增加率 - 營業收入增加率
  • 營業毛利指標 = 營業收入增加率 - 營業毛利增加率
  • 銷管費用指標 = 銷管費用增加率 - 營業收入增加率
  • 應付帳款指標 = 營業收入增加率 - 應付帳款增加率

現金轉換循環 = 平均銷貨天數 + 平均應收帳款收現天數 - 平均應付帳款付款天數
